# Transient-cryospheric-subsidies-mask-tipping-risks-beneath-dryland-greening

This repository contains the code and model implementations used in the study:
"Transient cryospheric subsidies mask tipping risks beneath dryland greening".

## Overview

This study develops a deep-learning framework to quantify the role of the Asian Water Tower (AWT) in sustaining downstream dryland vegetation across five major endorheic basins in Central Eurasia.

The study quantifies:

1. historical mountain-derived remote influences;
2. ecological subsidies provided by the AWT;
3. future vegetation trajectories under three SSP scenarios;
4. vegetation tipping dynamics under continued warming.

## Dataset:
The analysis combines multiple publicly available datasets.

Because several datasets are redistributed under their original licenses, this repository does not include all raw datasets. Users should download the original products from the corresponding providers and place them into the data/ directory following the folder structure above.

Processed datasets required to reproduce the figures are provided where licensing permits.
